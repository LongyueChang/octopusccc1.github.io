github提交

摘要:如何提交项目到github仓库

正文:


你现在已经有一个github账号了了

现在你需要做的是创建一个远程仓库



Start a project(也就是创建仓库)



第一栏填写你的仓库名称

Description栏写你对这个仓库的描述

然后选择确定，仓库就建好了



首先右击你的项目，如果不出意外的话会有Git bash here字样出现，点击，这时候会进入终端



然后进入你的项目里面



将你的远程仓库克隆过来

由于你什么都没有写所以你的远程仓库应该就是一个空的文件夹



把你的项目放进去

在终端中输入 git add .

“.”代表着所有，所以不要忘记写了



输入 git commit -m “提交了一个项目”

不要忘记描述你的文件哦^ ^



输入git push -u origin master 就提交成功了

萌新的教程，大佬看见了勿喷！！！
