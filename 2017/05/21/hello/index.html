创建博客

摘要:有关于如何创建博客的一篇日志

正文:


1.安装git;

2.安装Node环境;

3.得有一个github账号;

4.在github里得有一个仓库，而这个仓库就是你博客的域名

5.创建仓库之后得验证邮箱;

6.安装hexo ,这是一个基于hexo搭建的博客;

因为墙的存在,所以最好用淘宝镜像来下载$ npm install -g cnpm –registry=https://registry.npm.taobao.org

这窜代码相当于下载依赖？？脚手架？？$ cnpm install -g hexo-cli

正式下载 $ cnpm install hexo –save

检查一下下载是否成功$ hexo -v

初始化一下!$ hexo init

搭建完成！$ cnpm install

创建日志 $ hexo new “hello”

开启本地服务器浏览 $ hexo s -g

设置用户名$ git config –global user.name “你的用户名”

设置邮箱$ git config –global user.email “你的邮箱”

发布博客$ hexo d -g
